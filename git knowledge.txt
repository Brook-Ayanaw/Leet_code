git init --this initialize a folder to track the rep named git
git status --shows tracked and untracked files 
git -add --to stage is for tracking 
git commit -m "message" --this is when we are commiting or taking a snapshoot of the staged files 
git status --short m index.html M index.html --here when there is a non-commited change it will show s then name of the file 
git commit -a -m "Updated index.html with a new line" this will commit all staged files
git log --this shows the list of commits 
git command -help -  See all the available options for the specific command
git help --all -  See all possible commands
